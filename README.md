## Getting and Running FETTER

1. Make sure you have Java 8 (or newer) installed ([link](https://github.com/ojdkbuild/ojdkbuild))
2. Download the [FETTER.jar](https://github.com/OE-FET/FETTER/raw/master/FETTER.jar) file
3. Run the jar file.
